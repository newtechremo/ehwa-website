# 홈페이지 문구 수정 사항

> 11~24번 홈페이지 본문 내의 문구 수정 요청입니다.

---

## 11번: 메인 문장 변경

**수정 항목**: "복잡한 의료기관 ~ 신청해주세요" 문장 변경

**변경 후**:
```
장벽 없는 병원 이용,
장애인 의료기관 이용편의 지원사업과 함께라면 가능합니다.
```

---

## 12번: 서비스 소개 문장 수정 및 줄바꿈

**변경 후**:
```
병원 예약부터 진료, 귀가까지 병원 이용 전 과정에서 필요한 도움을 제공하는 서비스입니다.
수어통역사를 포함한 전담 인력이 1:1 맞춤형 서비스를 제공합니다.
```

---

## 13번: 이동·동행지원 문장 수정

**변경 후**:
```
혼자 이동하기 어려울 때 동행합니다.
휠체어 이용, 환복, 검사 중 자세 유지 등 신체 활동을 지원합니다.
```

---

## 14번: 행정절차 지원 문장 수정

**변경 후**:
```
서류작성, 키오스크(무인기기) 등 복잡한 행정절차를 도와드립니다.
필요한 경우 지역사회 복지자원과 연결해 드립니다.
```

---

## 15번: 의사소통 지원 문장 수정

**변경 후**:
```
수어통역이 필요할 때, 원내 전담 수어통역사가 전 진료 과정을 함께합니다.
글자판(필담), 그림판(ACC) 등 다양한 의사소통 방식을 활용합니다.
```

---

## 16번: 맞춤형 진료 지원 문장 수정

**변경 후**:
```
장애 유형 및 정도, 이동 동선을 고려하여 진료 일정과 대기 절차를 조정합니다.
진료 후 복약 지도 및 다음 내원 절차를 안내해드립니다.
```

---

## 17번: '최소 3일 전~' 문장 수정

**변경 후**:
```
원활한 지원을 위해 최소 3일 전까지 신청해 주시기를 권장합니다.
```

---

## 18번: Step 1 서비스신청 본문 문장 수정

**변경 후**:
```
편의지원 서비스를 신청합니다.
```

---

## 19번: Step 2 사전 상담

**변경 후**:
```
상담을 통해 필요한 도움을 확인하고 서비스 예약을 확정합니다.
```

---

## 20번: Step 3 병원 내원

**변경 후**:
```
요청하신 서비스와 함께 진료를 받습니다.
```

---

## 21번: Step 4 귀가 지원

**변경 후**:
```
진료 후 복약지도 및 다음 서비스 예약을 도와드립니다.
```

---

## 22번: 버튼 문구 변경

**수정 항목**: "편의지원 신청하기" 문구 변경

**변경 후**:
```
서비스 신청하기
```

**비고**: 상단과 통일

---

## 23번: 섹션 삭제

**삭제 대상**: "신청 후 진행과정" 섹션 전체 삭제

삭제할 내용:
- 신청 후 진행과정
- 1. 전화연락 - 신청하시면 ~
- 2. 이용확정 - (영상)전화 상담 후 ~

**비고**: 신청서에 진행 과정이 적혀있고, 카카오톡이나 전화는 과정 설명이 될 것 같아서 문구를 수정하면 좋을 것 같습니다.

---

## 24번: 문구 삭제

**삭제 대상**: "※ 토요일 진료는 ~" 내용 삭제
